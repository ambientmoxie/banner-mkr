Disclaimer:

This website has been created to facilitate the setup of traditional HTML5 animated banners. The output is a very basic HTML structure with a minimal GSAP timeline. This tool is a work in progress.

How to use:

  • Click and drag to change the order of blocks/frames.
  • Edit the timeline by clicking the "add" or "delete" buttons.
  • Download your HTML banner when you are done.
